# Simple Chat App (work in progress)

A real-time chat application built with Angular, focused on scalable frontend architecture, SSR, reactive programming, and WebSocket communication.


## Tech Stack

### Frontend
- **Angular 19**
- **TypeScript**
- **RxJS**
- **HTML5**
- **CSS3**

### Angular Ecosystem
- **Angular Router**
- **Reactive Forms**
- **HttpClient**
- **Angular SSR**

### Backend Communication
- **REST API**
- **WebSocket**
- **Proxy Configuration**

### Server
- **Node.js**
- **Express.js**

### Utilities
- **ngx-cookie-service-ssr**
